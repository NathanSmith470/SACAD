SACAD
Smith's Airfoil Computer Aided Design, used to simplilfy multiple services for individuals who work with airfoils

==================================================================================================================
The Windows executable file (exe) is in the compressed zip folder. It should be entirely standalone. But be warned because it may be very slow  and heavy on your ram. I'd recommend just running the .py file and installing tkinter.

The other three files are the raw Python code and the .ico file needed for the gui.
